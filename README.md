Python Practice

Small projects while learning Python fundamentals.

calculator.py - Command-line calculator that performs addition, subtraction, multiplication, and division based on user input. Includes input validation for invalid operators and division-by-zero handling, plus a replay loop that lets the user perform multiple calculations without restarting the program.

guessing_game.py — Number guessing game where the program picks a random number and the player guesses higher/lower until correct or out of attempts. Includes an attempt counter, a limit on total guesses, and a replay loop to play again without restarting.
